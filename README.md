## ⚡ Quick Start / Panduan Cepat

### 1️⃣ Prepare Ingredients (Persiapan)
- Windows 10/11
- [Python 3.10+](https://www.python.org/downloads/)
- Telegram Account

### 2️⃣ Get Your Credentials (ID & Token)
1. Chat with [@BotFather](https://t.me/BotFather) to create a bot and get the **API Token**.
2. Chat with [@userinfobot](https://t.me/userinfobot) to get your **Chat ID**.

### 3️⃣ Setup Environment (Konfigurasi)
Copy `.env.example` to `.env` and fill the variables:
```env
TELEGRAM_TOKEN=12345678:ABCDEFG...
TELEGRAM_CHAT_ID=987654321
```

### 4️⃣ Launch System (Jalankan)
```bash
# Install dependencies
pip install -r requirements.txt

# Start the bot
python spy_bot.py
```
